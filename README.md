# Pastel Crumbs Website

A bakery website built using HTML, CSS, and PHP.

Features:
- Home page with bakery introduction
- Menu page for desserts and cakes
- Services page
- Order page
- Contact page with Google Maps

The website was hosted online using InfinityFree.

Live Website:[Pastel Crumbs](https://pastelcrumbs.great-site.net)
